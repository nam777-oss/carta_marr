# 🌸 Flor Interactiva - Proyecto de Amor

Una hermosa aplicación web interactiva que presenta una flor animada con mensajes de amor y efectos de typing profesionales.

## ✨ Características

### 🌺 Página Principal
- **Flor animada CSS**: Hermosa flor creada con CSS puro
- **Animaciones hover**: La flor cambia de color y rota al pasar el cursor
- **Loading profesional**: Indicador de carga elegante sin spinner
- **Responsive design**: Se adapta perfectamente a todos los dispositivos

### 💬 Página de Mensaje
- **Efecto typing**: Mensaje que aparece letra por letra
- **Brillitos animados**: Pequeños brillitos rosados que flotan en el fondo
- **Diseño minimalista**: Interfaz limpia y profesional
- **Animación de cursor**: Cursor parpadeante que desaparece al terminar

### 💖 Página "Te Amo"
- **Typing animado**: "Te amo" aparece con efecto de máquina de escribir
- **Corazón SVG animado**: Corazón que se dibuja solo y luego se rellena
- **Fondo elegante**: Gradiente dorado suave
- **Transiciones suaves**: Animaciones fluidas entre elementos

## 🚀 Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible
- **CSS3**: 
  - Gradientes y animaciones avanzadas
  - Flexbox para layouts responsivos
  - Keyframes para animaciones personalizadas
  - Media queries para diseño responsive
- **JavaScript ES6+**:
  - Manipulación del DOM
  - Efectos de typing dinámicos
  - Control de navegación entre páginas
  - Generación dinámica de brillitos

## 🎯 Flujo de Usuario

1. **Inicio**: Usuario ve la flor con instrucción "Presiona la flor"
2. **Click**: Aparece loading "Cargando" por 2.5 segundos
3. **Mensaje**: Página con mensaje de amor que aparece con typing effect
4. **Brillitos**: Pequeños brillitos rosados aparecen durante el mensaje
5. **Regresar**: Click en "← Regresar" lleva a página "Te amo"
6. **Te Amo**: Texto aparece con typing + corazón se dibuja automáticamente

## 🛠️ Instalación y Uso

1. **Descarga**: Clona o descarga los archivos del proyecto
```bash
git clone [tu-repo-url]
cd flor
```

2. **Ejecutar**: Abre `index.html` en tu navegador preferido
   - No requiere servidor local
   - Funciona directamente desde el sistema de archivos

3. **Personalizar**: 
   - Edita `script.js` para cambiar los mensajes
   - Modifica `styles.css` para ajustar colores y animaciones
   - Personaliza `index.html` para agregar más contenido
